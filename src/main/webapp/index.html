About

This is part of a project I made during my internship at Vilnius Gediminas Technical University in Lithuania.

The technologies I used for this webapp are :

  • Jetty for the REST resources of the server.
  • MongoDB for the database.
  • JQuery for the Javascript programming.
  • Bootstrap, an elegant and easy to use CSS framework.

This project was made by Aymeric Giraudet
